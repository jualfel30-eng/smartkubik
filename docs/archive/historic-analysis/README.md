# Historic analysis (archive)

Esta carpeta archiva **snapshots analíticos del proyecto en distintos momentos** entre noviembre 2025 y marzo 2026: análisis de arquitectura, índices viejos del repo, reportes ejecutivos puntuales, evaluaciones ROI/gaps, y patrones de código de fases iniciales.

## Diferencia con `completed-roadmaps/`

- **`completed-roadmaps/`**: documentos que **planificaban** features que después se construyeron.
- **`historic-analysis/`** (esta carpeta): documentos que **describían** o **analizaban** el estado del proyecto en un momento puntual. Son fotografías, no planes.

## Cuándo consultar

- Querés entender cómo lucía el proyecto en noviembre 2025 vs hoy.
- Buscás contexto de una decisión arquitectural histórica.
- Hacés arqueología de patrones de código que evolucionaron.
- Necesitás ver el estado de un análisis específico (ROI restaurante, comparativa multi-país, etc.).

## Cuándo NO consultar

- Si necesitás el estado actual → `docs/wiki/` y `system-map.md` son la fuente viva.
- Si vas a implementar algo nuevo → `docs/wiki/patterns/` y `CLAUDE.md`.
- Si necesitás un índice del proyecto → `CLAUDE.md` raíz cumple esa función hoy.
