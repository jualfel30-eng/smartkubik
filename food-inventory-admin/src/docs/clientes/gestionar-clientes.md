# Cómo Gestionar Clientes, Proveedores y Contactos

## ¿Dónde está el CRM?

**Menú lateral:** Ventas y Marketing → CRM (`/crm`)

**Pestañas disponibles:**
- **Todos** — Todos los contactos
- **Clientes** — Solo clientes individuales y empresas
- **Proveedores** — Solo proveedores
- **Pipeline** — Embudo de ventas (oportunidades)

---

## Crear un contacto

1. Haz clic en **"+ Nuevo"**
2. Selecciona el tipo:
   - **Individual** — Persona natural (cliente)
   - **Empresa** — Persona jurídica (cliente)
   - **Proveedor** — Proveedor del negocio
   - **Empleado** — Trabajador del negocio
3. Llena los datos: nombre, RIF, contacto, dirección
4. Si es proveedor: configura condiciones de pago (crédito, métodos)

> **Auto-creación:** Los proveedores y empleados crean perfiles adicionales automáticamente (perfil de Supplier o EmployeeProfile).

---

## Programa de Lealtad

SmartKubik clasifica a tus clientes automáticamente en 4 niveles basados en su historial de compras:

| Nivel | Criterio | Color |
|-------|----------|-------|
| 🥉 **Bronce** | Clientes nuevos o poco frecuentes | Marrón |
| 🥈 **Plata** | Compran regularmente | Gris |
| 🥇 **Oro** | Buenos clientes (top 5-20%) | Dorado |
| 💎 **Diamante** | Mejores clientes (top 5%) | Azul |

**Cómo se calcula:** El sistema evalúa 3 factores:
- **Recencia** — ¿Cuándo fue su última compra? (50% del peso)
- **Frecuencia** — ¿Cuántas veces ha comprado? (30%)
- **Monto** — ¿Cuánto ha gastado en total? (20%)

La clasificación se actualiza automáticamente cada vez que se consulta la lista de clientes.

## Pipeline de Ventas

Si llevas seguimiento de oportunidades de negocio:

1. Ve a la pestaña **Pipeline**
2. Configura las **etapas** en Settings (ej: Lead → Contactado → Propuesta → Negociación → Cerrado)
3. Crea oportunidades y muévelas entre etapas
4. Vista **Kanban** (tarjetas) o **Tabla**

## Problemas comunes

### "Busco un cliente pero no aparece"

La búsqueda busca en: nombre, apellido, nombre de empresa, número de cliente, RIF, y datos de contacto. Si usas solo números del RIF (sin letra ni guión), también debería encontrarlo.

### "¿Un contacto puede ser cliente Y proveedor?"

Sí. Al editar el contacto, cambia el tipo a uno que incluya ambos roles. El sistema mantiene los datos de ambos perfiles.

### "¿Cómo veo el historial de compras de un cliente?"

Abre el contacto → pestaña **Transacciones**. Puedes filtrar por fecha, producto, monto, y ver estadísticas (total gastado, frecuencia, productos favoritos).
