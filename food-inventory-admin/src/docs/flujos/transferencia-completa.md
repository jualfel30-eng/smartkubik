# Transferencia entre Almacenes

## El flujo en 4 pasos

### 1. Crear → solo se guarda, no mueve nada
### 2. Aprobar → autorización, no mueve nada
### 3. Despachar → **AQUÍ se descuenta stock del origen** (no se puede deshacer)
### 4. Recibir → **AQUÍ se suma stock al destino**

> **Regla clave**: Después del despacho no se puede cancelar. Si algo sale mal, crea una transferencia en sentido contrario.

## ¿Qué pasa si lo que llega no coincide?

Si recibes menos de lo enviado, el sistema detecta la diferencia automáticamente y la marca como **discrepancia**. Puedes agregar fotos de evidencia.

## Express: todo en un clic

El botón **"Enviar Ahora"** hace todos los pasos de una vez (crear → solicitar → aprobar → preparar → despachar). Útil para transferencias urgentes.
